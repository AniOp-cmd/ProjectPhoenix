# mp-importvehicles
Import Vehicle Pack - NoPixel Inspired - Credit to Obtaizen

## Installation
- Drag and drop into `[resources]` folder
- Ensure `mp-importvehicles` in server.cfg
- Open vehicles.txt and paste into `qb-core/shared/vehicles.lua`
